# Calendar Extractor — Android App

> Extract → Preview → Add  
> Version 1.0.0 (build android)

## Overview

Extract calendar events from natural language text anywhere. Calendar Extractor uses AI to parse event details and add them to your calendar with a simple workflow.

**Device compatibility**: Android 8.0+

## Requirements

- Android 8.0 (API level 26) or higher
- Google Play Services
- Calendar app installed

## Installation

1. Download the APK file
2. Enable "Install from unknown sources" in Settings
3. Open the APK file to install
4. Grant calendar permissions when prompted

## How to Use

- **Share Intent**: Share text from any app to Calendar Extractor
- **Main Activity**: Open the app directly to enter event text
- **Widget**: Add a home screen widget for quick access (coming soon)

### Basic Workflow

1. Select or enter text describing an event (e.g., "Team meeting tomorrow at 2pm")
2. The app extracts event details automatically
3. Review the preview and confirm to add to your calendar

## Privacy & Data

- **Privacy-first**: No raw text is stored unless you explicitly opt-in to improve results
- **Local processing**: Event extraction happens on-device when possible
- **Support**: support@example.com
