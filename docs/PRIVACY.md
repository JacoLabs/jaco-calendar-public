# Privacy Policy

Last updated: November 8, 2025

## Overview

Calendar Extractor respects your privacy and is committed to protecting your personal information.

## Data Collection

- **Event Text**: Processed locally or sent to our API for parsing
- **Calendar Data**: Only accessed with your explicit permission
- **Usage Analytics**: We do not collect usage analytics

## Data Storage

- Event text is processed in real-time and not stored permanently
- No personal information is retained after processing
- All processing happens on-device or through secure API calls

## Third-Party Services

- Calendar integration uses your device's native calendar APIs
- No data is shared with third parties without your consent
- Optional LLM processing may use external services (configurable)

## Your Rights

You have the right to:
- Access your data
- Request data deletion
- Opt-out of any data collection
- Control which services process your data

## Security

- All API communications use HTTPS encryption
- No persistent storage of personal information
- Calendar permissions are requested only when needed

## Contact

For privacy concerns, contact: support@example.com

## Changes to This Policy

We may update this privacy policy from time to time. We will notify you of any changes by posting the new policy on this page.
