# Calendar Extractor — iOS App

> Extract → Preview → Add  
> Version 1.0.0 (build ios)

## Overview

Extract calendar events from natural language text anywhere. Calendar Extractor uses AI to parse event details and add them to your calendar with a simple workflow.

**Device compatibility**: iOS 14.0+

## Requirements

- iOS 14.0 or later
- iPhone or iPad
- Calendar access permissions

## Installation

1. Download from TestFlight or App Store
2. Install the app
3. Grant calendar permissions when prompted
4. Configure your preferred calendar in Settings

## How to Use

- **Share Extension**: Share text from any app to Calendar Extractor
- **Main App**: Open the app to enter event text
- **Siri Shortcuts**: Create custom shortcuts for quick event creation

### Basic Workflow

1. Select or enter text describing an event (e.g., "Team meeting tomorrow at 2pm")
2. The app extracts event details automatically
3. Review the preview and confirm to add to your calendar

## Privacy & Data

- **Privacy-first**: No raw text is stored unless you explicitly opt-in to improve results
- **Local processing**: Event extraction happens on-device when possible
- **Support**: support@example.com
