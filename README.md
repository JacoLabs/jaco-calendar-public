# Jaco Calendar

> Transform text into calendar events with AI  
> Version 1.0.0

## Overview

Jaco Calendar is a cross-platform tool that extracts calendar events from natural language text. Using AI-powered parsing, it automatically identifies event details like dates, times, locations, and descriptions, then adds them to your calendar with a simple workflow.

**Available on**: Browser Extension, Android, iOS

## Features

- **Natural Language Processing**: Describe events in plain text
- **Multi-Platform**: Works on web browsers, Android, and iOS
- **Privacy-First**: Local processing when possible, no unnecessary data storage
- **Smart Extraction**: Automatically detects dates, times, locations, and more
- **Calendar Integration**: Seamlessly adds events to your existing calendar apps

## Quick Start

Choose your platform:

### Browser Extension
- **Supported Browsers**: Chrome, Firefox, Edge (version 88+)
- **Installation**: Load the extension from `browser-extension/` folder

### Android

### iOS
- **Requirements**: iOS 14.0+
- **Installation**: TestFlight or App Store
- **Usage**: Share extension or main app interface
- [Full Documentation](ios/README.md)

## How It Works

1. **Input**: Select or enter text describing an event
   - Example: "Team meeting tomorrow at 2pm in Conference Room A"
2. **Extract**: AI automatically parses event details
3. **Review**: Preview the extracted information
4. **Confirm**: Add to your calendar with one tap

## Privacy & Security

- **Privacy-first design**: No raw text stored unless you opt-in
- **Local processing**: Event extraction happens on-device when possible
- **Minimal permissions**: Only requests necessary calendar access
- **Transparent**: Open source and auditable
