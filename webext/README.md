# Calendar Extractor — Browser Extension

> Extract → Preview → Add  
> Version 1.0.0 (build webext)

## Overview

Extract calendar events from natural language text anywhere. Calendar Extractor uses AI to parse event details and add them to your calendar with a simple workflow.

**Device compatibility**: Chrome, Firefox, Edge

## Requirements

- Modern web browser (Chrome 88+, Firefox 89+, Edge 88+)
- Internet connection for API access

## Installation

1. Download the extension package
2. Open your browser's extension management page
3. Enable "Developer mode"
4. Click "Load unpacked" and select the webext folder
5. The extension icon will appear in your toolbar

## How to Use

- **Context Menu**: Right-click selected text and choose "Create Calendar Event"
- **Extension Popup**: Click the extension icon to manually enter event text
- **Keyboard Shortcut**: Configure in browser settings (optional)

### Basic Workflow

1. Select or enter text describing an event (e.g., "Team meeting tomorrow at 2pm")
2. The app extracts event details automatically
3. Review the preview and confirm to add to your calendar

## Privacy & Data

- **Privacy-first**: No raw text is stored unless you explicitly opt-in to improve results
- **Local processing**: Event extraction happens on-device when possible
- **Support**: support@example.com
